### Team 1: BrainStormers (3 students)
**Topic**: Dynamic Pricing on E-Commerce Platforms with DRL
**Research Paper**: https://arxiv.org/pdf/1912.02572
**Implementation**: DQN-based pricing agent for simulated e-commerce environment
**Complexity**: Medium - requires economics understanding + RL
**Key Deliverables**:
- E-commerce pricing environment
- DQN agent with experience replay
- Comparison with baseline strategies (fixed, random pricing)
- Revenue optimization results

### Team 2: Model Minds (3 students)
**Topic**: Deep Q-Network (DQN)
**Research Paper**: https://arxiv.org/abs/1312.5602
**Implementation**: Classic DQN on CartPole-v1
**Complexity**: Medium - foundational algorithm
**Key Deliverables**:
- DQN implementation from scratch
- Experience replay buffer
- Target network mechanism
- Component analysis (with/without target network)

### Team 3: PhuRaTan (3 students)
**Topic**: Using Reinforcement Learning to play Ultimate Tic-Tac-Toe
**Research Paper**: https://arxiv.org/pdf/2212.12252
**Implementation**: Q-learning or SARSA for Ultimate Tic-Tac-Toe
**Complexity**: Medium-High - complex game rules
**Key Deliverables**:
- Ultimate Tic-Tac-Toe game implementation
- Q-learning agent with self-play
- Strategy analysis
- Win rate against baselines

### Team 4: Q-Team (3 students)
**Topic**: Exploration of Reinforcement Learning to SNAKE
**Research Paper**: https://cs229.stanford.edu/proj2016spr/report/060.pdf
**Implementation**: DQN for Snake game
**Complexity**: Medium - reward shaping challenges
**Key Deliverables**:
- Snake game environment
- DQN agent
- Training progression
- Survival time and score improvements

### Team 5: The Crown (3 students)
**Topic**: Q-Learning Fundamentals
**Research Paper**: https://arxiv.org/abs/2108.02827

**Complexity**: Low-Medium - fundamental concepts
**Key Deliverables**:
- Q-learning on FrozenLake, Taxi, CliffWalking
- Hyperparameter analysis
- Convergence comparison
- Policy visualization
